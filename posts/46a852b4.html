描述统计：统计量

在描述行统计中，除了统计图表，统计量可以定量的描述数据。本章介绍常见的统计量：均值、方差、标准差、中位数、四分位数、变异系数、IQR、极差、偏度、峰度、Q-Q 图。以及分类汇总函数 aggregate 。

1 描述统计：统计量

定性变量：频数、比例

定量变量：集中信息、波动信息、形状信息

1.1 集中信息

1.1.1 样本均值

对于数据 x1, x2, ⋯ , xnx_1,\ x_2,\ \cdots,\ x_nx1​, x2​, ⋯, xn​ 的样本均值为：

xˉ=x1+x2+⋯+xnn=1n∑i=1n xi\bar{x} = \frac{x_1 + x_2 + \cdots + x_n}{n} = \frac{1}{n}\sum\limits_{i = 1}^n\ x_i xˉ=nx1​+x2​+⋯+xn​​=n1​i=1∑n​ xi​

缺点：均值容易受到极端值影响。

1.1.2 中位数

样本中位数是对原始数据排序，位于数据中心的数据。

xMED=x(n+12)or(x(n2)+x(n2+1))/2x_{MED} = x_{(\frac{n+1}{2})}\quad or\quad (x_{(\frac{n}{2})} + x_{(\frac{n}{2} + 1)})/2 xMED​=x(2n+1​)​or(x(2n​)​+x(2n​+1)​)/2

假设有定量数据 x1, x2, ⋯ , xnx_1,\ x_2,\ \cdots,\ x_nx1​, x2​, ⋯, xn​ ，定义排序后的数据 x(1)≤x(2)≤⋯≤x(n)x_{(1)} \leq x_{(2)} \leq \cdots \leq x_{(n)}x(1)​≤x(2)​≤⋯≤x(n)​ 。

优点：稳健，不容易受到极端值影响。

1.2 波动信息

1.2.1 方差/标准差

样本方差：反映了所有数据离中心的平均的平方距离

s2=1n−1∑i=1n (xi−xˉ)2s^2 = \frac{1}{n-1}\sum\limits_{i=1}^n\ (x_i - \bar{x})^2 s2=n−11​i=1∑n​ (xi​−xˉ)2

样本标准差：近似反映了所有数据离中心的平均距离

s=s2=1n−1∑i=1n (xi−xˉ)2s = \sqrt{s^2} = \sqrt{\frac{1}{n-1}\sum\limits_{i=1}^n\ (x_i - \bar{x})^2} s=s2​=n−11​i=1∑n​ (xi​−xˉ)2​

1.2.2 变异系数

变异系数 Coefficient of Variation (CVCVCV) ：比较不同单位的数据的离散程度；比较单位相同而平均数相差较大的数据组资料的差异程度。

CV=sxˉ×100CV = \frac{s}{\bar{x}} \times 100 CV=xˉs​×100

1.2.3 四分位间距/极差

四分位差 (Inter - Quartile Range, IQRIQRIQR) ：中间 50%50\%50% 数据的范围。

IQP=Q3−Q1IQP = Q_3 - Q_1 IQP=Q3​−Q1​

极差 ：所有数据的范围

range=x(n)−x(1)range = x_{(n)} - x_{(1)} range=x(n)​−x(1)​

1.3 形状信息

1.3.1 偏度

总体偏度的定义

Skewness=E (X−μσ)3Skewness = E\ (\frac{X - \mu}{\sigma})^3 Skewness=E (σX−μ​)3

样本偏度的定义

g1=n(n−1)(n−2)⋅s3∑i=1n (xi−xˉ)3g_1 = \frac{n}{(n-1)(n-2)\cdot s^3} \sum\limits_{i=1}^n\ (x_i - \bar{x})^3 g1​=(n−1)(n−2)⋅s3n​i=1∑n​ (xi​−xˉ)3

偏度判断：

  • 对称：Skewness = 0
  • 右偏：Skewness > 0
  • 左偏：Skewness < 0

1.3.2 峰度

峰度反应了峰部的尖度，总体峰度的定义：

Kurtosis=E (X−μσ)4−3Kurtosis = E\ (\frac{X - \mu}{\sigma})^4 - 3 Kurtosis=E (σX−μ​)4−3

样本峰度：

g2=n(n+1)(n−1)(n−2)(n−3)⋅s4∑i=1n (xi−xˉ)4−3⋅(n−1)2(n−2)(n−3)g_2 = \frac{n(n+1)}{(n-1)(n-2)(n-3)\cdot s^4}\sum\limits_{i=1}^n\ (x_i - \bar{x})^4 - 3\cdot \frac{(n-1)^2}{(n-2)(n-3)} g2​=(n−1)(n−2)(n−3)⋅s4n(n+1)​i=1∑n​ (xi​−xˉ)4−3⋅(n−2)(n−3)(n−1)2​

峰度判断：

  • 正态分布：Kurtosis = 0

  • 峰度越大，高峰分布：两端数据多（厚尾），中间数据少，且集中在均值附近。

  • Kurtosis > 0 ，与正态分布相比是高峰

1.3.4 Q-Q 图

Q-Q Plot ：对于观测数据 x1, x2, ⋯ , xnx_1,\ x_2,\ \cdots,\ x_nx1​, x2​, ⋯, xn​ 对其排序后得到 x(1)≤x(2)≤⋯≤x(n)x_{(1)} \leq x_{(2)} \leq \cdots \leq x_{(n)}x(1)​≤x(2)​≤⋯≤x(n)​ ，它们的概率为：

P(X≤x(j))=jn≈j−0.375n+0.25P(X \leq x_{(j)}) = \frac{j}{n} \approx \frac{j - 0.375}{n + 0.25} P(X≤x(j)​)=nj​≈n+0.25j−0.375​

然后取出标准正态分布的分位数点 F−1(j−0.375n+0.25)F^{-1}(\frac{j - 0.375}{n + 0.25})F−1(n+0.25j−0.375​) 。绘制散点图：

(F−1(j−0.375n+0.25), x(j))(F^{-1}(\frac{j - 0.375}{n + 0.25}),\ x_{(j)}) (F−1(n+0.25j−0.375​), x(j)​)

若大致为线性关系/直线，则可以认为 xix_ixi​ 满足近似正态。
